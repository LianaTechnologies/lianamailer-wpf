== Description ==

LianaMailer for WPForms is an integration plugin for a [WPForms](https://wordpress.org/plugins/wpforms-lite/) plugin which integrates LianaMailer into WPForms.

= Contribute and translate =

LianaMailer for WPForms plugin is developed and supported by Liana Technologies Oy. Join our crew at GitHub or contact using other means.

== Frequently Asked Questions ==

= Where can I find LianaMailer documentation and user guides? =

For help setting up and configuring Liana Mailer, please refer to [LianaMailer support site](https://support.lianatech.com/lianamailer.html).

= Where can I report bugs? =

Report bugs on our [GitHub repository](https://github.com/LianaTechnologies/lianamailer-wpf/issues).

You can also notify us via our support channels.

= Where can I request new features, themes, and extensions? =

Request new features and extensions and vote on existing suggestions on our official customer support channels. Our product team regularly review requests and consider them valuable for product planning.

= LianaMailer for Gravity Forms is awesome! Can I contribute? =

Yes, you can! Join in on our [GitHub repository](https://github.com/LianaTechnologies/lianamailer-wpf)

= Where can I find REST API documentation? =

REST API documentation of LianaMailer product is available at [our support site](https://support.lianatech.com/lianamailer/settings/rest-api.html).
